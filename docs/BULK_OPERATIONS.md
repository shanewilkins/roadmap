## CLI Commands

```bash
# Bulk operations
roadmap bulk validate --comprehensive
roadmap bulk health-report --output report.json
roadmap bulk backup --destination ./backup

# Data operations
roadmap export issues --format xlsx
roadmap export analytics --format json
```
