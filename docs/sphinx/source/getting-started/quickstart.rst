What is Roadmap CLI?
====================

Roadmap CLI is an enterprise-grade command-line tool for managing project
roadmaps with GitHub integration and powerful analytics.

Basic Workflow
==============

1. Initialize a new roadmap
2. Create projects
3. Define milestones
4. Add and track issues
5. Sync with GitHub
6. Generate reports

Example Session
===============

.. code-block:: bash

    # Initialize a new roadmap
    roadmap init my-project

    # Create a project
    roadmap project create "Q1 Planning"

    # View status
    roadmap status

    # Get help
    roadmap --help
