How do I sync with GitHub?
===========================

Use the ``roadmap git`` commands to sync with GitHub.

How do I migrate from v.0.x?
============================

See the migration guide in Getting Started.

Where are my data files stored?
===============================

Data is stored in ``.roadmap/`` directory by default.

Can I use multiple projects?
=============================

Yes, each project is independent.

How do I backup my data?
========================

Backup the entire ``.roadmap/`` directory.
