Command Reference
================================================================================

Complete reference for all Roadmap CLI commands.

.. note::

   This is a stub. Complete implementation coming in v.0.7.0.

General Usage
=============

.. code-block:: bash

    roadmap [OPTIONS] COMMAND [ARGS]...

Global Options
==============

- ``--version`` - Show version
- ``--help`` - Show help

Commands
========

Project Commands
----------------

- ``roadmap project create`` - Create a new project
- ``roadmap project list`` - List all projects
- ``roadmap project view`` - View project details
- ``roadmap project update`` - Update a project
- ``roadmap project delete`` - Delete a project

Milestone Commands
------------------

- ``roadmap milestone create`` - Create a milestone
- ``roadmap milestone list`` - List milestones
- ``roadmap milestone view`` - View milestone details
- ``roadmap milestone update`` - Update a milestone
- ``roadmap milestone close`` - Close a milestone

Issue Commands
--------------

- ``roadmap issue create`` - Create an issue
- ``roadmap issue list`` - List issues
- ``roadmap issue view`` - View issue details
- ``roadmap issue update`` - Update an issue
- ``roadmap issue done`` - Mark issue as done

Other Commands
--------------

- ``roadmap status`` - Show overall status
- ``roadmap init`` - Initialize a new roadmap
- ``roadmap health`` - Check system health

See the API documentation for detailed parameter information.
