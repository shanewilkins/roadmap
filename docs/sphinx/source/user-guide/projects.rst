Creating a Project
==================

Projects define high-level planning for your organization or team.

See :doc:`commands` for the ``project create`` command reference.

Project Structure
=================

Each project can contain:

- Milestones (time-based deliverables)
- Issues (individual work items)
- Metadata (owner, status, dates)
