================================================================================
Managing Issues
================================================================================

Issues represent individual work items, tasks, or features in your roadmap.

.. note::

   This is a stub. Complete implementation coming in v.0.7.0.

Creating Issues
===============

Issues are the atomic units of work in Roadmap CLI.

See :doc:`commands` for the ``issue create`` command reference.

Issue Status
============

Issues can have various statuses:

- **todo** - Not started
- **in_progress** - Currently being worked on
- **done** - Completed
- **closed** - Archived or cancelled

GitHub Integration
==================

Issues can be synced with GitHub issues.

See :doc:`../user-guide/github` for details.
