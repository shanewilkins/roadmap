What is the Demo Project?
=========================

The demo project (in the ``demo-project/`` directory) is a complete,
working example of a Roadmap CLI setup. It demonstrates:

- Project structure
- Best practices
- Real-world workflows
- GitHub integration

Running the Demo
================

To explore the demo project:

.. code-block:: bash

    cd demo-project
    cat README.md              # Read the demo instructions
    ls -la                     # Explore the structure
